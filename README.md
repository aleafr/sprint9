# Sprint 9 — Priorização de Hipóteses e Análise de Teste A/B

Projeto desenvolvido durante o bootcamp de Data Analytics da TripleTen.

## Objetivo

Analisar hipóteses de negócio para aumentar a receita de uma loja online e avaliar os resultados de um teste A/B.

## Descrição

Neste projeto foram realizadas:
- priorização de hipóteses utilizando frameworks ICE e RICE;
- análise exploratória dos dados;
- estudo de métricas de conversão;
- análise do tamanho médio dos pedidos;
- identificação e remoção de anomalias;
- análise estatística de teste A/B;
- interpretação de resultados e tomada de decisão baseada em dados.

## Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
- Jupyter Notebook
